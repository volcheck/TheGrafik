# TheGrafik
A web application to create a duty sheet, hardcoded for 4 team members daytime and nighttime shifts.
Это написанная для Python+Flask тулза для ручного создания графика дежурств врачей. База данных на SQLite3. Фронт стандартный (html5/CSS/JS).
Это НЕ программа для генерации графика! Его надо настрадать вручную и экспортировать в ворд.
